Flutter Sign Up & Log In App

A simple Flutter app demonstrating user registration, login, and navigation. Users can sign up with an email and password, log in, and access a home page. This app stores users locally in memory (no database).

Features

Sign Up with email and password
Log In with registered credentials
Simple Home Page showing logged-in user
Navigation between Login, Sign Up, and Home pages
Error handling for invalid login or empty fields

Project Structure
lib/
 ├─ main.dart        # Main entry point & all pages (Login, Sign Up, Home)
pubspec.yaml          # Project dependencies

All Flutter code is inside lib/main.dart.
